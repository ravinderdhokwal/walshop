## Steps to run the application
1. Clone the repository.
2. Navigate/change directory to the backend.
3. Run command *"npm install"* to download all the required dependencies.
4. Rename ".env_sample" file to ".env" and set all the required variables.
5. Run command *"npm run build"* to build the files which will be seen in *dist* folder.
6. Run command *"npm run start* to start the application.
